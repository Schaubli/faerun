Loading Data
  1. Select a database.
  2. Select a (molecular) fingerprint.
  3. Select a variant. Each variant is the product of a PCA dimensionality reduction from original fingerprint space to 3D. After projection, the data was digitized into two different resolutions: low (250 x 250 x 250) and high (500 x 500 x 500).
  4. After this step, the positional data is loaded (all data points have the same colour.
  5. It's strongly suggested that you now also select a map, colour coding the loaded data. The colour gradient (dark blue to magenta) always equals low to high.
  6. You can now select data points by double clicking them.
Filtering Data
  1. Data filtering allows to explore the inside of the point cloud.
  2. Data can be filtered by using Depth Cutoff. Setting a minimum depth from the camera, thus slicing the point cloud.
  3. Data can alos be filtered by Color.
Details (1)
  1. A data point (or bin) can be selected by double clicking it. Once selected, it appears in the selected bins list in the HUD.
  2. A selected item has three functions. Clicking the 'x' button removes the bin from the selected list. Clicking the button below the 'x' button centers the view on this bin. Clicking the item anywhere else, opens the detail view in a new browser tab.
  3. The details view loads the content of a bin, or if the bin has less than 10 compounds, the content of the bin and the 26 surrounding bins.
  4. Data points representing compounds are connected by the minimum spanning tree.
  5. Data points contained within the selected bin are strongly saturated (colourful) while those selected from surrounding bins have muted colours.
Details (2)
  1. Compounds within a bin (or cluster of bins) can also be explored by displaying them as a list.
  2. The button "Download As CSV" downloads a CSV containing the displayed data.
Loading (Projecting) a Custom Set of Molecules
  1. A custom set of molecules can be projected to the currently loaded space by clicking "Project" in the main menu.
  2. Choose a color for the projected data points.
  3. Choose a name for the set.
  4. Enter (or paste) a set of SMILES strings in the textarea, separated by new lines and click "PROJECT".
  5. The (currently selected) fingerprint is computed for the entered molecules.
  6. The PCA model (matrices) associated with each fingerprint is (are) stored and applied to the calculated fingerprints.
  7. The projected points are probably hidden by the original data. Since the two filter sliders are always associated with the main data layer (SureChEMBL), they can be used to bring the newly projected data within view.
KNN Search
  1. Once a custom projection is loaded, make sure it is selected in the LAYERS list and all filters are off.
  2. In the main menu, click "KNN SEARCH".
  3. Choose a number of nearest neighbours (k) to serach for (for each projected compound).
  4. This step might take up to a couple of minutes, depending on the amount of projected compounds, the parameter k and the speed of your computer.
  5. Once the KNN search has finished, select SureChEMBL again in order to be able to select data points from this set (or layer).
